Corporate Landing Page Documentation

Introduction

Thank you for purchasing our theme. This documentation will guide you through the setup and customization of the theme.

File Structure

The file structure is organized as follows:

Project_Root/
├── HTML/
│   ├── assets/
│   │   ├── css/
│   │   │   ├── bootstrap.min.css
│   │   │   ├── style.css (Your custom code)
│   │   │   └── responsive.css
│   │   ├── js/
│   │   │   ├── bootstrap.bundle.min.js
│   │   │   └── main.js
│   │   ├── img/ (All images go here)
│   │   └── fonts/ (Icon fonts like FontAwesome)
│   ├── services/
│   │   ├── web-design.html (Details page for Web Design)
│   │   ├── web-development.html (Details page for Web Development)
│   │   └── seo-optimization.html (Details page for SEO Optimization)
│   ├── index.html
│   └── about.html
├── Documentation/
│   └── index.html (How to use the theme)
└── Licensing/ (License info for assets)

Adding New Service Pages

To add a new service page, you can duplicate one of the existing service pages (e.g., web-design.html) in the HTML/services/ directory. Update the content, images, and links as needed. Remember to also add a "Learn More" button in the services section of index.html to link to your new service page.

Case Studies Integration

Case study content is organized in the Project_Root/Documentation/case-studies.txt file. You can create individual case study pages in a dedicated folder (e.g., HTML/case-studies/) similar to how service pages are handled.

To integrate a new case study:

  1. Create a new HTML file (e.g., case-study-1.html) within HTML/case-studies/.
  2. Populate the HTML file with content from case-studies.txt or your custom content. Ensure to maintain consistency with the theme's design.
  3. Link to this new case study page from your main index.html (e.g., from a dedicated "Case Studies" section) or any other relevant page.
  4. Remember to adjust relative paths for CSS and JS files in your new case study HTML file (e.g., ../../assets/css/style.css if placed inside HTML/case-studies/).

Customization

To customize the theme, you can edit the style.css file located in HTML/assets/css/. All the custom styles for the theme are in this file.

You can also change the images in the HTML/assets/img/ directory to your own images.

Fonts

The theme uses Google Fonts. The fonts used are 'Montserrat' for headings and 'Open Sans' for body text. You can change the fonts by editing the link in the <head> section of the HTML files.
